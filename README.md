# NoSneakAnimation

A simple client-side Fabric mod that removes the smooth sneaking camera transition. When you press sneak, your camera instantly moves to the crouched position.

## Versions

| Branch | Minecraft Version | Download |
|--------|-------------------|----------|
| [26.2](https://github.com/pit-mod/NoSneakAnimation/tree/26.2) | 26.2 | [Releases](https://github.com/pit-mod/NoSneakAnimation/releases) |
| [1.21.x](https://github.com/pit-mod/NoSneakAnimation/tree/1.21.x) | 1.21 – 1.21.x | [Releases](https://github.com/pit-mod/NoSneakAnimation/releases) |

## Features
- Instant camera snap when sneaking/uncrouching
- No configuration needed — works out of the box
- Client-side only, no server required
- No dependencies (only Fabric Loader)

## Installation
1. Install [Fabric Loader](https://fabricmc.net/) for your Minecraft version
2. Drop the `.jar` into your `.minecraft/mods` folder
3. Launch the game
