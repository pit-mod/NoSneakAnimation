# NoSneakAnimation (1.21.x)

A simple client-side Fabric mod for **Minecraft 1.21.x** that removes the smooth sneaking camera transition. When you press sneak, your camera instantly moves to the crouched position.

## Features
- Instant camera snap when sneaking/uncrouching
- No configuration needed — works out of the box
- Client-side only, no server required
- No dependencies (only Fabric Loader)

## Installation
1. Install [Fabric Loader](https://fabricmc.net/) for Minecraft 1.21.x
2. Drop `NoSneakAnimation-1.0.jar` into your `.minecraft/mods` folder
3. Launch the game

## Building
```shell
./gradlew build
```
The compiled jar will be in `build/libs/`.
